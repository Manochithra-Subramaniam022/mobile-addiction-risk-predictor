# Mobile Addiction Risk Predictor

## Overview
This project predicts mobile addiction risk based on user behavior such as screen time, app usage, and data consumption.

## Features
- Data cleaning and preprocessing
- Exploratory Data Analysis (EDA)
- Feature engineering (Screen Intensity Score)
- Machine Learning model (Random Forest)
- Model evaluation and insights

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Results
The model achieved high accuracy and identified screen time and app usage as key predictors of addiction risk.
